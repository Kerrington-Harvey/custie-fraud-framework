# Custie Fraud Operations Framework

End-to-end fraud operations framework designed and implemented from scratch as the founding Fraud Operations Manager at Custie, a B2B/B2C pet-care platform. Built without third-party fraud tooling.

## Key Highlights

- Built full fraud ops framework from zero - no third-party tooling
- KYC/KYB onboarding with 0 fraud incidents
- Reduced time-to-decision by 33% through data minimization control
- 100% TPR | 0% FPR | 0% decline rate across full pilot
- Threat modeled across New Account Fraud and Account Takeover vectors pre-launch
- Designed for analyst-level execution with Fraud Operations Manager decisioning authority
